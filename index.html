PSApi

Features

Movies

Fetch List

Returns a JSON response of all Star Wars movies, along with their opening crawls and anonymous comment counts. Movies are sorted by release date in ascending order.

End Point:

https://may-psapi.herokuapp.com/movies/list.php

Sample Response:
{
movies: 
[
    {
        "title": "A New Hope",
        "episode_id": 4,
        "opening_crawl": "It is a period of civil war. Rebel....",
        "release_date": "1977-05-25",
        "comment_count": "2"
    }
]
}

Comments

Fetch List

Returns a JSON response of movie comments stored in a database. Comments are made anonymously but api will fetch user's ip address and date & time of posting.

End Point:

https://may-psapi.herokuapp.com/comments/list.php

Sample Response:
{
"records": [
    {
        "film_name": "Attack of the Clones",
        "comment": "Dummy comment 2",
        "user_ip": "10.63.255.214",
        "posted_at": "2019-04-18 14:40:35"
    },
    {
        "film_name": "Return of the Jedi",
        "comment": "this comment is from php input three",
        "user_ip": "::1",
        "posted_at": "2019-04-16 19:00:10"
    }]
}

Add Comment

Allows user to create comments anonymously for any movie.

Input requirements:

  • Only movie Id and user comment allowed.
  • Comments are restricted to 500 characters.
  • Commenter's IP address will be stored

Comment create can be tested using Postman

End Point:

https://may-psapi.herokuapp.com/comments/create.php

Sample Input:

{
    "film_id":"5",
    "comment":"Dummy comment 2"
}

Characters

Fetch List

Returns a JSON response of all characters in a specified star wars movie. Characters in a movie can be filtered by gender and sorted by permitted parameters.

End Point: Base URL

https://may-psapi.herokuapp.com/characters/list.php

Fetch all characters in the star wars universe.

Sample Response:

{
    "count": 87,
    "next": "https://swapi.co/api/people/?page=2",
    "previous": null,
    "results": [
        {
            "name": "Luke Skywalker",
            "height": "172",
            "mass": "77",
            "hair_color": "blond",
            "skin_color": "fair"...
        }
    }

List characters for specific movie

End Point:

https://may-psapi.herokuapp.com/characters/list.php?id=3

Sample Response:

{
    "title": "Return of the Jedi",
    "film_id": "3",
    "characters": [
    {
        "name": "Luke Skywalker",
        "gender": "male",
        "height": "172"
    },
    {
        "name": "C-3PO",
        "gender": "n/a",
        "height": "167"
    }]
}

Filter Characters

Use predefined parameters to filter listed characters based on gender

Filters:

  • Male
  • Female
  • Hermaphrodite

Undefined parameters return characters with gender set as n/a

Response will include information about the total height of filtered characters in centimeters and inches

End Point:

https://may-psapi.herokuapp.com/characters/list.php?id=3&gender=female

Sample Response:

{
    "characters": [
        {
            "name": "Leia Organa",
            "gender": "female",
            "height": "150"
        },
        {
            "name": "Mon Mothma",
            "gender": "female",
            "height": "150"
        }
    ],
    "Total Height": {
        "cm": 300,
        "feet": 9.8425196850393704
    }
}

Sort Characters

Use predefined parameters to sort listed characters based on gender, name and height

Sort order must be specified as ascending or descending

Sort Parameters:

  • gender
  • name
  • height

Order Parameters:

  • asc
  • desc
End Point:

https://may-psapi.herokuapp.com/characters/list.php?id=3&sort=gender&order=asc
